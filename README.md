# Java简易swing通信录
***
###主要功能：
1. 用户的注册与登录;
2. 实现简单的通信录管理，能够输入通信录卡片(实现增、删、改等功能)，能够分组，并能够实现按好友的任意信息进行查询；
3. 实现界面：Windows界面，功能用菜单形式，输入界面用文本框形式，数据保存在数据库中。
